# ttool

gogen  代码生成工具


### 功能：

- go orm  orm生成

```
代码自动生成工具，包括golang app项目生成，orm生成

Usage:
  gogen [command]

Available Commands:
  help        Help about any command
  orm         根据 db.yml 文件自动生成orm
  version     show version

Flags:
  -h, --help   help for gogen

Use "gogen [command] --help" for more information about a command.
```

### 安装

`make install`
